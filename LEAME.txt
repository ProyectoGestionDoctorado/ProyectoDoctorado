Este programa contiene la arquitectura base para desarrollar aplicaciones web con JavaServerFaces y PrimeFaces.

Para abrir el proyecto se requiere:

1. NetBeans 8.0.2 o superior
2. GlassFish 4.1 o superior
3. MySQL
4. Primeface 5.0
5. JDK 1.7

En MySql crear un usuario  llamado proyectobase y que tenga permisos para acceder a la base de datos del mismo nombre (proyectobase). El password de proyectobase debe ser 123456
